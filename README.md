# Macvim + Janus

## Files

Open/Close the NerdTree

```sh
:NERDTreeToggle
```

Refresh new files created for Command + T

```sh
:ClearCtrlPCache
```

## Editing modes

```sh
i = Insert
```

```sh
v = Visual
```

```sh
d = Delete
```

## Moving

```sh
$ = End of the line
```

```sh
0 = Beginning of current line
```
